# 기능 상세 요구사항 관련 메뉴 요청 부서 1차/2차 중복 참조
1 정보 수정 권한 제한 회원 기본정보 수정은 인증팀 주담당, 센터장/전략기획 팀장은 보조 수정 가능 정회원/준회원 상세 매칭·전략 2차#38 수정 권한 분리 (권한자 구체화)
2 최근 수정 정보 시각화 수정된 칸에 한 달간 색깔/테두리 표시 → 매니저가 변경사항 인지 후 프로필도 수정 가능 정회원/준회원 상세 매칭·전략 —
3 전체 변경 이력 추적 현재 특정 정보만 변경이력 남음 → 모든 칸 수정 시 최근 변경 표시 (별도 변경이력 페이지 불필요, 칸 자체에 표시) 정회원/준회원 상세 매칭·전략 2차#39 변경 이력 표시 (범위 확대: 특정→전체 칸)
4 탈회 CS관리 중 상태 신설 탈회 언급 후 탈회금 미확정 상태를 표시할 새 상태값 필요 (소보원·소송 진행 중 포함) 정회원 상태관리 매칭·전략 —
5 보류 기간 경과 알림 임시보류·약정보류 설정 후 일정 기간 경과 시 담당 매니저에게 알림 (자동 상태변경 ❌, 알림만) 정회원 상태관리 매칭·전략 2차#28 만료일 자동 알림 (보류 상태까지 확장)
6 상태 변경 시 사유 기록 보류·교제 등 상태 변경 시 변경 사유·일시·처리자 기록 → 인수인계 시 히스토리 확인 가능 정회원 상태관리 매칭·전략 —
7 컨택 예약 캘린더 알림 매니저가 특정 회원에 대해 컨택 예정일 설정 → 해당 날짜에 알림 발송 (재가입 컨택, 보류 해제 확인 등) 정회원/준회원 관리 매칭·전략 2차#21 전화 예약 알림 + 2차#49 매칭 캘린더 (통합 확장)
8 자동 상태변경 금지 원칙 임시보류·외부교제 등 자동 해제/만료 불가 → 수동 관리 유지, 알림 기반 관리만 적용 정회원 상태관리 매칭·전략 —
9 만료일 경과 시 프로필 발송 차단 만료 상태 미변경(활동)이어도 만료일자 경과 시 프로필 발송 + 만남 등록 자동 차단 정회원 미팅현황 매칭·전략 2차#29 만료 시 미팅등록 차단 (프로필 발송까지 확장)
10 만료 후 후기 등록 허용 만료일 경과 후에도 미팅 후기 등록은 가능하도록 별도 처리 정회원 미팅현황 매칭·전략 —
11 프로필 거절 → 미응답 변경 프로필 발송 후 무응답 시 '거절' 대신 '미응답/응답지연' 상태로 처리 (해외 회원 등 장기 지연 대비) 정회원 매칭관리 매칭·전략 2차#47 프로필 수락 기한 정책 (거절 대안 구체화)
