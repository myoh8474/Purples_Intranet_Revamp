## 1. 모듈 개요

| 항목 | 내용 |
|------|------|
| **모듈명** | 매칭 관리 (MN-310 ~ MN-313) |
| **핵심 목적** | 등급별 매칭 필터링 → 프로필 발송 → 안심번호 발급 → 미팅 → 교제 관리 전체 파이프라인 |

## 2. 등급별 매칭 필터링 정책

### 2.1 +1단계 검색 룰 (일반/준전문직)

| 회원 등급 | 검색 가능 범위 | 차단 범위 |
|----------|-------------|----------|
| 브론즈 | 브론즈 ~ 실버(+1) | 골드 이상 전체 |
| 실버 | 실버 ~ 골드(+1) | 다이아 이상 |
| 골드 | 골드 ~ 다이아(+1) | 블랙/시크릿 |
| 다이아 | 다이아 ~ 블랙(+1) | 시크릿 |

> ⚠️ **시스템 하드코딩**: 매니저가 해제/우회 불가

### 2.2 전문직 블라인드

| 검색 주체 | 대상 | 허용 여부 |
|----------|------|:--------:|
| 전문직 남성 → 모든 여성 | 전체 | ✅ |
| 골드↑ 여성 → 전문직 남성 | 전문직 | ✅ |
| 브론즈/실버 여성 → 전문직 남성 | 전문직 | 🚫 원천 차단 |

### 2.3 VVIP 무제한 오픈

블랙/시크릿 등급 및 기타-자산가, 특수 프로모션 회원 → 등급 제한 없이 전체 매칭 풀 접근

---

## 3. 매칭 대상자 검색 (MN-310)

**검색 필터:**
- 필수: 성별(자동 반대), 나이 범위, 지역
- 선택: 학력, 직업, 종교, 키, 자산, 혼인이력, 프로그램/등급

**자동 필터링 (시스템 적용):**
1. 등급 +1단계 제한
2. 매칭 제외 조건 체크 (소송중, 보류, 블랙리스트)
3. 기매칭 이력 중복 제거
4. 프로필 Lock 상태 회원 제외

---

## 4. 매칭 요청/수락/거절 (MN-311)

**프로세스:**
1. 매칭 매니저가 후보 선정 → 프로필 발송
2. 양측 회원에게 프로필 전달 (안심번호 자동 발급)
3. 72시간 내 응답 대기

| 응답 | 시스템 처리 |
|------|-----------|
| 쌍방 수락 | 매칭 성사 → 미팅 일정 조율 |
| 한쪽 거절 | 거절 사유 기록 → 다음 후보로 |
| 72시간 무응답 | 자동 거절 (RL-011) → 양측 알림 |

---

## 5. 안심번호 관리 (MN-312)

| 이벤트 | 처리 |
|--------|------|
| 미팅 수락(OK) | 050x 자동 발급 + 양측 안내 |
| 교제 거절/종료 | 자동 회수(파기) |
| 탈회 완료 | 전체 안심번호 회수 |

---

## 6. 교제 관리 (MN-313)

| 상태 | 진입 조건 | 기간 제한 | 전환 |
|------|----------|----------|------|
| 임시교제 | 2번째 만남 예정 | 21일 Lock | 활동 복귀 or 교제 전환 |
| 교제 | 3번째 만남 예정 | - | 결렬 → 활동 / 결혼 → 결혼예정 |
| 외부교제 | 외부 이성 교제 통보 | - | 결렬 → 활동 |
