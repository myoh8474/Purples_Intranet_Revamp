## 3. 외부 연동 API (5종)

### 3.1 공공데이터 및 마이데이터 스크래핑 API

- **우선순위**: 🔴 Critical
- **목적**: 수동 위임장 기반 대행 발급 업무 → 전면 자동화
- **기능**: 간편인증(카카오/PASS) 동의 → 서류 자동 수합 → 진위 자동 검증
- **효과**: 4대 서류 Pass 시 [인증중] → [활동] 자동 전환, 매칭 Lock 해제

| 서류명 | API/발급처 | 방식 |
|--------|-----------|------|
| 혼인관계증명서 | 대법원 전자가족관계등록시스템 | 스크래핑 |
| 가족관계증명서 | 대법원 전자가족관계등록시스템 | 스크래핑 |
| 졸업증명서 | 학교 발급 시스템 | 스크래핑/직접등록 |
| 건강보험자격득실확인서 | 국민건강보험공단 | API |
| 소득금액증명원 | 국세청 홈택스 | 스크래핑 |
| 주민등록초본 | 정부24 API | 스크래핑 |

- **기술스택**: PASS/카카오 간편인증 SDK, Codef/바로빌 API, S3, PDF.js

### 3.2 안심번호(가상번호) 발급 API

- **우선순위**: 🟠 High
- **목적**: 개인정보보호법 대응 + 분쟁 방지
- **기능**: 미팅 수락 → 050x 자동 발급 → 교제 종료 시 자동 회수
- **기술스택**: KT 050, SK 안심번호 등 통신사 API

### 3.3 마케팅 채널 웹훅(Webhook) 연동 API

- **우선순위**: 🔴 Critical
- **목적**: 가망 고객 DB 자동 정제 + 영업 충돌 방지
- **기능**: 카카오/네이버/메타/CPA 실시간 인서트 → 중복 필터링 → 스팸 차단
- **기술스택**: Webhook Receiver (REST API), 채널별 API 키 관리

### 3.4 통합 커뮤니케이션 API (알림톡/SMS/메일)

- **우선순위**: 🟠 High
- **목적**: 수동 메시지 발송 → 룰 엔진 연동 100% 자동화
- **주요 알림**: 부재중 5회 이관, 보류 만료, 72시간 무응답 거절, 임시교제 21일 촉구, 미팅 확정 안내
- **기술스택**: 카카오 알림톡 API, SMS(부스트/인포뱅크), SendGrid/AWS SES

### 3.5 전자계약 및 전자서명 API

- **우선순위**: 🟠 High
- **목적**: 외부 의존 계약/환불 파이프라인 → 인트라넷 내재화
- **기능**: 가입 시 전자서명, 탈회 시 전자서명, 서명 원본 DB 저장 + 감사 추적
- **효과**: 서명 완료 → 계약 시작일 Lock → 탈회 시 위약금 자동 산정
- **기술스택**: 모두싸인/이싸인/카카오페이 인증서, 공동인증서 연동
