# PRD: 회원분배 관리 모듈

> 버전: 1.0 | 날짜: 2026-05-12 | 담당: 인포팀, 관리자

---

## 1. 모듈 개요

| 항목 | 내용 |
|------|------|
| **모듈명** | 유입DB 분배 관리 (MN-500 ~ MN-502) |
| **핵심 목적** | 마케팅 채널 DB 자동 유입 → 중복 탐지 → 매니저 균등 분배 |
| **관련 테이블** | TB_ASSOCIATE, TB_DISTRIBUTE |

---

## 2. 분배 대상 및 자격

| 구분 | 조건 | 비고 |
|------|------|------|
| 분배 대상 DB | [컨텍전] 상태 준회원 | 미배정 신규 DB만 |
| 분배 제외 | [진행불가], [소스외], [블랙리스트] | 대기열 자동 제외 |
| 수신 매니저 | 현재 재직 상담팀 매니저 | 퇴사/휴직/인증팀/매칭팀 제외 |
| 상한선 | 일일 **20건** / 월간 **200건** | 초과 시 자동 스킵 |
| 재컨텍 DB | [기간만료(재컨텍)] + 쿨타임 완료 | 출처 태그별 분배 |

---

## 3. 분배 방식

### 3.1 자동 분배 (라운드로빈)
- 활성 매니저 순서대로 1건씩 균등 배정
- 먼저 유입된 DB부터 우선 분배
- 동일 지사 소속 매니저에게만 분배 (지사 격리)
- 가중치 옵션: 신입 매니저 절반, 베테랑 1.5배

### 3.2 수동 분배
- 팀장급 이상만 실행 가능
- 미분배 회원 선택 → 매니저 지정 → 실행
- 상한선 초과 시 경고 팝업 (관리자 강제 배정 옵션)

---

## 4. 중복 DB 수거

| 단계 | 처리 |
|------|------|
| 1 | 유입 DB 휴대폰 번호 → 기존 전체 DB 대조 |
| 2-A (고유) | 정상 분배 대기열 등록 |
| 2-B (중복) | [소스외] 자동 전환 + 기존 매니저 알림 |
| 3 | 중복 수거 이력 자동 기록 (원본 연결 보존) |

---

## 5. 재분배(담당 변경) 정책

| 항목 | 규칙 |
|------|------|
| 변경 사유 | **필수 입력** (미입력 시 차단) |
| 승인 | 일반 매니저 → 팀장 승인 / 팀장↑ → 즉시 |
| 실적 귀속 | 변경 전 → 기존 매니저 / 변경 후 → 신규 매니저 |
| 쿨타임 | 동일 DB 7일 이내 2회 이상 재분배 불가 |

---

## 6. 분배 후 미컨텍 통제

| 경과 시간 | 처리 |
|----------|------|
| 48시간 | 담당 매니저 미컨텍 경고 알림 |
| 72시간 | 팀장 긴급 알림 + 재분배 검토 자동 지정 |

|------|----------|:--------:|:------:|:--------:|
| [정회원 만료] | 약정/기간/자동만료 | 1순위 | 90~180일 | 14일 |

| [장기상담 만료] | 장기상담 90일 경과 | 3순위 | 60일 | 14일 |

## 9. 분배 실패 예외 처리

| 상황 | 처리 |
|------|------|
| 가용 매니저 0명 | 미분배 대기 목록 보류 + 관리자 긴급 알림 |
| 채널 연동 장애 | 5분 간격 3회 재시도 → 실패 시 관리자 알림 |
| 스팸/허위 DB | 자동 차단 + 차단 내역 기록 |
