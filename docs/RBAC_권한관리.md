## 1. 부서(팀) 및 역할 정의

| 코드 | 부서(팀) | 구성원 | 주요 업무 |
|------|----------|--------|----------|
| `MATCH` | 매칭팀 | 매칭센터장, 매칭상담매니저 | 매칭 대상자 검색, 프로필 발송, 약속 배정, 미팅 관리 |
| `CONSULT` | 상담팀 | 지사총괄, 지사본부장, 상담매니저 | 준/정회원 상담, 매출 입력, 컨택 관리 |
| `CERT` | 인증팀 | 인증팀장 | 서류인증, 회원정보 수정, 전산 생성 |
| `SALES` | 영업팀 | 영업본부장, 영업팀장 | 영업 전략, 매출 관리, 프로모션 |
| `FIN` | 재무팀 | 재무팀장 | 매출 확인/수정, 정산, 환불 처리 |
| `EXEC` | 경영팀 | 경영본부장 | 전체 데이터 열람, 경영 통계, 리스크 관제 |
| `MKT` | 홍보마케팅 | 홍보본부장, 홍보팀사원 | 마케팅 채널 관리, 유입DB 분석, 광고 운영 |
| `CS` | 고객만족실 | 본부장 | 클레임 접수/처리, 탈회 진행, CS 관리 |
| `LEGAL` | 법무팀 | 법무팀장 | 소송 관리, 블랙리스트, 계약 검토 |
| `PLAN` | 전략기획팀 | 전략기획팀장 | 정책 수립, 업무 프로세스 기획 |
| `CEO` | 최고관리자 | 회장님 | 최종 승인, 강제보류, 탈회 최종 결재 |
| `ADMIN` | 시스템관리자 | 개발부장, 기획팀장 | 권한 설정, 시스템 설정, 전체 관리 |

## 4. 데이터 범위 제한 (RLS)

| 역할 | 조회 범위 |
|------|----------|
| `MGR` | 본인 담당 회원만 |
| `TL` | 소속 팀 전체 |
| `FIN` | 전 브랜드 매출 데이터 |
| `MATCH` | 전체 정회원 (매칭 용도) |
| `CS` | 클레임/탈회 건만 |
| `HR` | 매니저 인사 데이터 |
| `EXEC` | 전체 (브랜드 코드별 필터) |
| `ADMIN` | 전체 (제한 없음) |

---

## 5. 특수 권한 규칙

1. **매출 수정**: `FIN` + `ADMIN`만 가능. 수정 시 변경 이력 자동 기록.
2. **쉐어링 비율 설정**: `TL` + `FIN` + `ADMIN`. 설정 후 실매출 자동 재계산.
3. **정산 승인**: `FIN`이 처리, `EXEC`가 최종 승인.
4. **회원 분배**: `TL` + `ADMIN`만 가능. 분배 로그 기록.
5. **탈회 처리**: `CS` 접수 → `TL` 승인 → `FIN` 환불 처리.
6. **경영/전략 메뉴**: `EXEC` + `ADMIN` 전용. 일반 직원 메뉴에 노출 안 함.
