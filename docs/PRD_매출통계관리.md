# 매출 및 통계 관리 모듈 PRD 개편 명세서

> 날짜: 2025. 05. 22 | 문서 성격: 브랜드 통합 및 데이터 정밀 정산을 위한 차세대 매출/통계 관리 명세

---

## 1. 개요 및 모듈 개편 방향

3개 브랜드(퍼플스/디노블/르매리) 통합 마스터 DB → **3분할 구조**:
- **재무/정산**: 실매출 기반 쉐어링 및 수당 정산
- **인사/평가**: 활동성 지표 + DB 회전율 중심 매니저 KPI 자동 산출
- **경영/전략**: 마케팅 채널별 LTV 분석 + 전사 리스크 관제

---

## 2. [재무/정산] 실매출 및 정산 관리

### 2.1 수납 구분

| 수납 구분 | 설명 | 트리거 |
|----------|------|--------|
| 일시불 | 당일 전액 완납 | 선배정 전산 자동 생성 |
| 계약금 | 일부 결제 | - |
| 중도금 | 잔여금 일부 | - |
| 잔금 | 최종 완납 | 선배정 전산 자동 생성 |
| 업그레이드 | 등급 상향 매출 | 별도 카테고리 |

### 입력 인터페이스 통제
- **1가입**: 준회원 창 [매출입력] → 매칭 매니저 선배정 연동
- **2가입**: 인트라넷 전용 매출창 Hard-lock

### 남성 수당 산정
- 전문직/재력가 체크박스 (본사 100억/지사 50억)
- → 인사 모듈 남성순위 + 수당 정산 즉시 전송

### 2.2 쉐어링 관리
- 산식: `입금액 × 설정 비율(%) = 실매출액` (N분할 가능)
- 매출 확정 시 각 매니저 실매출순위 자동 바인딩

### 2.3 전자계약 기반 자동 정산
- 서명 완료 → `contract_start_date` 하드코딩
- 탈회 시 → 위약금 자동 산출 → 정산팀 승인 (SLA 7영업일)

## 3. [인사/평가] 매니저별 KPI

### 3.1 전사 8대 랭킹
1. **매출순위**: 총 유입 가입비
2. **실매출순위**: 쉐어링 적용 후 금액
3. **성혼비순위**: 성혼 인센티브
4. **탈회순위**: 탈회 빈도 (리스크)
5. **남성순위**: 전문직/재력가 유치
6. **프로모션순위**: 이벤트 상품 판매
7. **컨택순위**: 콜수 + 컨택 성공
8. **환산미팅순위**: 미팅 전환 실적

→ 상담 1/2팀, 매칭팀, 지사별 필터 제공

### 3.2 활동성 지표
- 부재중 5회 or 14일 경과 → DB 자동 회수
- 장기상담 90일 초과 → 자동 이관
- 가입보류 30일 → 자동 회수
- 임시교제 21일 하드락

---

## 4. [경영/전략] 전사 영업 통계

### 4.1 상태값 6개 생애주기 그룹 (41개)
1. 준회원/영업 → 기한 초과 시 자동 전환
2. 결제/인증 → All-Pass 시 활동 전환
3. 정회원/활동 → 보류 만료 시 Roll-back
4. 교제 관리 → 21일 타임아웃
5. 만료/탈회 → 재가입 DB 풀 이관
6. 성혼/CS → 결혼예정 30일 미납 경고

### 4.2 마케팅 채널 효율
- 채널별 전환율 + LTV (`brand_code` 필수)
- **Welcome 통계**: 계약 → 첫 프로필 발송 소요기간
- 리스크 관제: 클레임 빈도 + 결혼예정 30일 초과

---

## 5. 기술 요구사항
- **UID 중심 통합**: N차 계약 매핑 (1가입/2가입/재가입 시계열)
- **RLS**: `brand_code` 기반 행 단위 보안
- **API 연동**: 인증 자동화 + 알림톡 자동 발송
