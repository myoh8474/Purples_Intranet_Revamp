---

## 1. 모듈 개요

| 항목 | 내용 |
|------|------|
| **모듈명** | 인증관리 (MN-400 ~ MN-402) |
| **핵심 목적** | 정회원 서류인증 자동화 — 스크래핑 API + 간편인증 기반 4대 서류 자동 수합/검증 |
| **관련 테이블** | TB_CERT_DOC, TB_REGULAR |

## 2. 4대 필수 서류

| 서류명 | 발급처 | 수합 방식 | 검증 방법 |
|--------|--------|----------|----------|
| 혼인관계증명서 | 대법원 전자가족관계등록시스템 | 스크래핑 | 자동 진위 검증 |
| 가족관계증명서 | 대법원 전자가족관계등록시스템 | 스크래핑 | 자동 진위 검증 |
| 건강보험자격득실확인서 | 국민건강보험공단 | API | 자동 진위 검증 |
| 소득금액증명원 | 국세청 홈택스 | 스크래핑 | 자동 진위 검증 |

**추가 서류:** 졸업증명서(학교), 주민등록초본(정부24)

---

## 3. 인증 프로세스

### 3.1 자동인증 경로
1. 회원에게 간편인증 요청 SMS/알림톡 발송
2. 회원 본인인증 (PASS/카카오)
3. 동의 항목 확인 (개인정보 수집)
4. 스크래핑 API 호출 → 서류 자동 발급
5. PDF 원본 저장 (S3/NAS)
6. 서류 진위여부 자동 검증
7. 4대 전체 Pass → **[인증중] → [활동] 자동 전환** + 매칭 Lock 해제

### 3.2 직접등록 경로
매니저가 서류 파일 직접 업로드 → 인증팀 수동 검증

---

## 4. 서류 반려 프로세스

| 단계 | 처리 | 기한 |
|------|------|------|
| 반려 처리 | 인증팀이 반려 사유 입력 | 즉시 |
| 자동 알림 | 담당 매니저에게 반려 사유 알림 자동 발송 | 즉시 |
| 보완 기한 | 매니저가 회원에게 서류 보완 요청 | **7일 이내** |
| 미보완 시 | 자동 가입 파기 처리 (RL-020) | 7일 경과 |

---

## 5. 최종 가입 거절 사유 (4대 사유)

1. 법적으로 가입이 불가한 사유
2. 회원 조건이 회사 가입 기준 미달
3. 회원 신원을 신뢰할 수 없는 경우
4. 필수 서류 작성/제출 거부

> 🔴 **서류 위조 적발 시** → 영구 가입 차단(블랙리스트) 등록

---

## 6. 예외 처리 대기열

API 조회 실패, 해외 대학, 특수 전문직 면허, 자영업자 매출 증빙 등
→ 인증팀 **수동 검토 Task 뷰**로 자동 이관 → 즉시 실패 처리하지 않음 (예외 보호)
